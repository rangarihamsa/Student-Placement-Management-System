# Student Placement Management System

## Project Overview

Student Placement Management System is a web-based application developed using HTML, CSS, and JavaScript. It helps manage student placement records, track placement status, search students, filter data, and display placement statistics.

---

## Features

- Add Student
- Edit Student Details
- Delete Student
- Search Student by Name
- Filter Students by CGPA
- Sort Students by CGPA
- Count Total Students
- Placement Status (Placed / Not Placed)
- Company Name Tracking
- Dashboard Analytics
  - Total Students
  - Placed Students
  - Not Placed Students
  - Average CGPA
- Export Student Data to CSV
- View Student Details
- Dark Mode
- Local Storage Support
- Responsive User Interface

---

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
4. Start adding student records.
